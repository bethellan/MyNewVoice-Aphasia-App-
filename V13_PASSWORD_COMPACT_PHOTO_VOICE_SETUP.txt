MyNewVoice v13 - Password-protected compact Photo / Voice Setup

Changes:
- Photo / Voice Setup is now behind the existing management password.
- Photo / Voice Setup now uses a section picker plus a collapsed item list and one selected-item control panel, rather than repeating full controls for every phrase.
- SAVE CHANGES closes the setup screen and returns to the main app.
- My People phrase images use a fixed circular crop.
- Other phrase images use a fixed square crop.
- Top menu images use a fixed square crop.
- Zoom images use a fixed rectangle crop.
- Shape options are no longer shown to the user.
- The photo picker accepts image files from the device. On iPhone/Android this allows the native photo/camera chooser where the browser supports it.
- The Main menu button inside submenus is smaller.
- Settings/theme/about icons are visible on the main screen only and hidden inside submenus.
